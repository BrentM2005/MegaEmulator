## About <a name="about"></a>


**MegaEmulator** is a Windows desktop app that was built for fans of gaming and emulation. It works out of the box as you can download
emulators and ROMs right in the app. It makes emulation easier and quicker to get started. It also makes it easy to manage all your ROMs
if you have games for several different platforms.


## Motivation <a name="motivation"></a>


Honestly, just built this app for fun and to learn C++/Qt. Although, I do plan to use it to manage my ROMs and emulators and hope others
will do the same. 


## Key Features <a name="key-features"></a>


- Download ROMs
- Download emulators
- Add your own ROMs
- Add your own emulators
- Add cover art for your games


## Getting Started <a name="getting-started"></a>


To get started you'll need to download our installer https://megaemulator.s3.us-east-2.amazonaws.com/MegaEmulator_Setup.exe/
Then, go through the installation setup.


Locally to setup for your own development, you'll need Qt. I used Qt Creator. Just clone this project
`git clone https://github.com/BrentM2005/MegaEmulator.git` then open the project in Qt Creator.  To build an executable
you will do build, then clean, then qmake, then build all. You will need the Release configuration. Go to projects,
build settings, active build configuration, and add release.  Then, if you want to make your own installer, I suggest 
using Inno Setup.
